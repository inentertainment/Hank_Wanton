# Сайт стихов — Хэнк Уэнтон

## Как опубликовать сайт (один раз)

1. Зайди на github.com и создай новый репозиторий (New repository).
   Назови его, например, `poems` (можно любое имя).
   Сделай его **Public**. Ничего лишнего не добавляй при создании.

2. Открой этот репозиторий в браузере. Нажми **Add file → Upload files**
   и перетащи туда ВСЕ файлы и папки из этой посылки (сохраняя структуру
   папок: `_poems`, `_layouts`, `_includes`, `assets` и т.д.). Внизу нажми
   **Commit changes**.

3. В репозитории открой **Settings → Pages** (в левом меню).
   В разделе "Build and deployment" выбери:
   - Source: **Deploy from a branch**
   - Branch: **main**, папка **/ (root)**
   Нажми **Save**.

4. Подожди 1–2 минуты. Обнови страницу Settings → Pages — вверху появится
   ссылка вида `https://твой-логин.github.io/poems/`. Это и есть твой сайт.

5. Открой файл `_config.yml` в репозитории (кнопка карандаша ✏️ для
   редактирования прямо на сайте github.com) и впиши в поля `url` и
   `baseurl` свой адрес, например:
   ```
   url: "https://твой-логин.github.io"
   baseurl: "/poems"
   ```
   Сохрани (Commit changes). Через минуту сайт пересоберётся.

## Как добавить новое стихотворение (когда угодно)

1. Зайди в репозиторий на github.com, открой папку **_poems**.
2. Нажми **Add file → Create new file**.
3. Назови файл так: `2026-07-12-nazvanie-stiha.md`
   (дата в начале обязательна, дальше — любое название латиницей, без пробелов).
4. Вставь в файл вот такой шаблон и замени текст:

   ```
   ---
   title: "Название стихотворения"
   date: 2026-07-12
   ---
   Первая строка стиха
   Вторая строка стиха

   Новая строфа начинается после пустой строки.
   ```

5. Нажми **Commit changes**. Через 30–60 секунд стихотворение само
   появится на сайте — в списке "Все стихи" и на главной странице.
   Ничего больше делать не нужно — ни список, ни ссылку "следующее
   стихотворение", ни поиск обновлять вручную не надо, всё считается
   автоматически.

## Важные детали

- Разрыв строки внутри стихотворения — это просто перевод строки.
  Пустая строка между строфами разделяет их визуально.
- Файлы можно загружать хоть каждый день, хоть раз в год — ограничений нет.
- Если сайт не обновился — подожди минуту и обнови страницу (Ctrl+F5).
  Прогресс сборки можно посмотреть во вкладке **Actions** репозитория.
- Тема (тёмная/светлая) переключается кнопкой ☾/☀ в шапке сайта и
  запоминается в браузере читателя.
